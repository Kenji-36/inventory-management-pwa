  - [ ] テストカバレッジレポートの自動生成

**推定工数**: 1-2日

**参考実装**:
```yaml
# .github/workflows/test.yml
name: Test & Deploy
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '18'
      - run: npm ci
      - run: npm test
      - run: npm run build
```
