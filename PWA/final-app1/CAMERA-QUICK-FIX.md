# カメラ許可の簡単な設定方法

## 🚀 最も確実な方法（3ステップ）

### ステップ1: Chrome設定を開く

**以下のURLをコピーして、ブラウザのアドレスバーに貼り付けてEnterキーを押してください：**

```
chrome://settings/content/camera
```

### ステップ2: localhostをブロックから削除

画面に「ブロック」というセクションがあります。

その中に `localhost:3000` または `http://localhost:3000` が含まれていたら：

1. その項目の右側にある **「︙」（3つの点）** をクリック
2. **「削除」** をクリック

### ステップ3: localhostを許可リストに追加

画面の上部に「許可」というセクションがあります。

1. **「追加」** ボタンをクリック
2. 以下を入力：
   ```
   http://localhost:3000
   ```
3. **「追加」** ボタンをクリック

### ステップ4: ページをリロード

1. アプリのタブに戻る（`http://localhost:3000/inventory`）
2. **F5キー** を押してページをリロード
3. **バーコードスキャン** ボタンをクリック
4. カメラが起動するはずです！

## ❓ よくある質問

### Q: アドレスバーの左側に🔒が見当たらない

**A:** URLの左端を探してください。以下のいずれかがあるはずです：
- 🔒（鍵マーク）
- ⓘ（情報マーク）
- 🌐（地球マーク）
- 🎥（カメラマーク - ブロック中）

### Q: 「カメラ」という項目が表示されない

**A:** まだカメラを使おうとしていないため、項目が表示されていない可能性があります。
→ 上記の「Chrome設定から直接変更」の方法を使ってください。

### Q: 「許可」を選択したのにカメラが起動しない

**A:** 以下を試してください：
1. ページを **Ctrl+Shift+R** で強制リロード
2. ブラウザを完全に終了して再起動
3. 他のアプリ（Zoom、Teamsなど）がカメラを使用していないか確認

以下のスクリーンショットを送ってください：

1. **ブラウザの画面全体**（アドレスバーが見える状態）
